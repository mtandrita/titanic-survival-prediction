## Tools Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Model
I used a Random Forest Classifier trained on cleaned features including:
- Passenger class
- Sex
- Age
- Fare
- Family aboard
- Embarkation port
